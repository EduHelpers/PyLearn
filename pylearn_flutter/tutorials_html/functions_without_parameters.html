Функции

Функции - это крайне удобный и востребованный инструмент для программиста. Мы с вами уже использовали многие функции, например print(), input(), list(), str(). Также методы, которые мы изучили, тоже являются функциями. Давайте разберёмся, для чего вообще они нужны и как создавать собственные функции.

Функции без параметров

Допустим нам нужно вывести несколько раз блок из символов "#", обычно это делается вот так:

for i in range(5):
    print("#"*20)
print()
for i in range(5):
    print("#"*20)
print()
for i in range(5):
    print("#"*20)

####################

####################

В коде мы использовали множественное повторение одних и тех же строк, что, согласитесь, выглядит не очень хорошо, и в большом проекте подобный код бы сильно мешал. Чтобы оптимизировать вывод как раз можно использовать функции. Зададим нашу первую функцию:

def print_str():
    for i in range(5):
        print("#"*20)
print_str()

Output:

####################
####################
####################
####################
####################

Функция задаётся ключевым словом def, после чего через пробел пишется её название и скобки сразу после него. В общем виде структура функции выглядит так:

def название_функции():
    .... #тут какой-то код, который функция выполняет

Вызов функции это её последующее использование в коде. Все функции, которые мы использовали, мы вызывали.

Важно! def задаёт функцию, но не вызывает её! Вызов должен быть строго после её объявления.

Также в функциях можно использовать другие функции, в том числе созданные нами:

def print_str():
    for i in range(5):
        print("#"*20)
def print_three_strs():
    for i in range(3):
        print_str()
        print()
print_three_strs()

Output:

####################
####################
####################
####################
####################

####################
####################
####################
####################
####################

####################
####################
####################
####################
####################

Название функций принято писать прописными буквами, разделяя слова символом _ . Регистр букв важен! Кроме того, название функций нельзя начинать с цифры. Также желательно передавать смысл функции в её названии, чтобы не запутаться.
