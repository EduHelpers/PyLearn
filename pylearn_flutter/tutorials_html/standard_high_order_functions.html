Стандартные функции высшего порядка

функция map

Функция map принимает на вход какую-то функцию и массив данных, применяет данную функцию к массиву и выводит обработанный массив. Например, функцией map можно преобразовывать массив строк в массив чисел:

    l = input().split()
    print(l)
    mapped_l = list(map(int, l))
    print(mapped_l)
    print(sum(mapped_l))

Input:

    1 2 3

Output:

    1 2 3
    ['1', '2', '3']
    [1, 2, 3]
    6

При этом map возвращает значения в неудобном для нас формате, его можно преобразовать в список через list().

функция filter

Функция filter отфильтровывает подходящие нам значения из массива, если переданная функция вернула True:

    l = [2, 17, 4, 8, 5, 12]
    def is_more_10(n):
        return n >= 10
    print(list(filter(is_more_10, l)))

Output:

    [17, 12]

Значение filter, как и map, тоже лучше преобразовать в list.

В целом это все основные функции высшего порядка, которые довольно часто применяются в программировании. Самое главное нарешать тесты для понимания работы этих функций.
