Словари

Словари это тоже крайне востребованный в программировании тип контейнера. В целом словари очень похожи на списки, с отличием в том, что обращение элементу происходит по специальному ключу. Кроме того, ключи и значения с словаре расположены не по порядку добавления! Также словари по-другому инициируются. Рассмотрим такую задачу: пусть у нас есть какие-то овощи и фрукты, и для каждого вида нам нужно хранить его массу на складе:

l = {"banana":3.14, "apple":7, "onion":0}
print(l["banana"])

Output:

3.14

Как мы видим, элементы словаря представляют собой пару ключ - значение. Ключом может быть любой неизменяемый тип данных, то есть сделать список ключом не получится. Значения могут быть любыми, в том числе список может быть значением:

l = {"banana":[1,2,3,7]}
print(l["banana"][3])

Output:

7

То есть сначала происходит обращение по ключу, потом по индексу списка, который является значением ключа.

Способов задать пустой словарь два либо через фигурные скобки, либо командой dict():

l = {}
r = dict()
print(l)
print(r)

Output:

{}
{}

Чтобы добавить новую пару ключ - значение в словарь, нужно просто задать значение по ключу:

l = {}
l["apple"] = 12
print(l["apple"])

Output:

12

Если обратиться по ключу, которого в словаре нет, возникнет ошибка:

l = {}
l["apple"] = 12
print(l["banana"])

Output:

Traceback (most recent call last):
  File ..., line 3, in <module>
    print(l["banana"])
KeyError: 'banana'

Методы словарей

Метод clear() очищает словарь:

l = {}
l["car"] = 12
print(l)
l.clear()
print(l)

Output:

{'car': 12}
{}

Метод copy()  возвращает копию словаря. Работает так же, как и со списками:

l = {"banana":3.14, "apple":7, "onion":0}
r = l.copy()
r["potato"] = 100
print(l)
print(r)

Output:

{'banana': 3.14, 'apple': 7, 'onion': 0}
{'banana': 3.14, 'apple': 7, 'onion': 0, 
    'potato': 100}

Метод items() возвращает пары (ключ, значение):

l = {"banana":3.14, "apple":7, "onion":0}
print(l.items())

Output:

dict_items([('banana', 3.14), ('apple', 7), 
    ('onion', 0)])

Метод keys() возвращает ключи в словаре:

l = {"banana":3.14, "apple":7, "onion":0}
print(l.keys())

Output:

dict_keys(['banana', 'apple', 'onion'])

Метод get(key, default) возвращает значение ключа, но если его нет, возвращает default (по умолчанию None):

l = {"banana":3.14, "apple":7, "onion":0}
print(l.get("banana"))
print(l.get("potato", "we don`t have potato"))

Output:

3.14
we don`t have potato

Метод pop(key, default) удаляет ключ и возвращает значение. Если ключа нет, возвращает default (по умолчанию бросает исключение):

l = {"banana":3.14, "apple":7, "onion":0}
print(l.pop("banana"))
print(l)
print(l.pop("potato", "we don`t have potato"))

Output:

3.14
{'apple': 7, 'onion': 0}
we don`t have potato

Метод setdefault(key, default) возвращает значение ключа, но если его нет, создает ключ со значением default (по умолчанию None):

l = {"banana":3.14, "apple":7, "onion":0}
print(l.get("banana"))
print(l.get("potato", 100))
print(l)

Output:

3.14
100
{'banana': 3.14, 'apple': 7, 'onion': 0}

Метод update([other]) обновляет словарь, добавляя пары (ключ, значение) из other. Существующие ключи перезаписываются. Возвращает None (не новый словарь!):

l = {"banana":3.14, "apple":7, "onion":0}
r = {"potato":100, "apple":24}
print(l)
l.update(r)
print(l)

Output:

{'banana': 3.14, 'apple': 7, 'onion': 0}
{'banana': 3.14, 'apple': 24, 'onion': 0, 
    'potato': 100}

Метод values() возвращает значения в словаре:

l = {"banana":3.14, "apple":7, "onion":0}
print(l.values())

Output:

dict_values([3.14, 7, 0])
