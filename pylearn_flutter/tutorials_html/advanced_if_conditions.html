Вложенный оператор

Внутри условного оператора можно использовать любые инструкции языка Python, в том числе и условный оператор. Получаем вложенное условия: после проверки одного условия программа переходит к проверке следующего(вложенного) условия. При этом вложенные блоки имеют больший размер отступа (+4 пробела для каждого следующего уровня).

if условие1:
    блок кода
else:
    if условие2:
        блок кода
    else:
        if условие3:
            блок кода
        ...

Рассмотрим программу, которая переводит стобалльную оценку в пятибалльную. Для ее реализации нужно воспользоваться вложенным условным оператором:

grade = int(input('Введите вашу отметку по 100-балльной системе: '))

if grade >= 90:
    print(5)
else:
    if grade >= 80:
        print(4)
    else:
        if grade >= 70:
            print(3)
        else:
            if grade >= 60:
                print(2)
            else:
                print(1)

В этом примере уровень вложенности настолько глубок, что код становится трудно понять.

Выбор из нескольких альтернатив – это обычное дело, здесь имеет смысл избегать глубокого вложения. Для этого в Python есть каскадный условный оператор.

Заметьте, что мы не могли написать сразу 5  if-ов каждый в отдельной строке, поскольку в таком случае было бы напечатано сразу несколько значений оценки.

Каскадный условный оператор

Если требуется проверить несколько условий, в языке Python используется каскадный условный оператор.

Синтаксис каскадного условного оператора имеет следующий вид:

if условие1:
    блок кода
elif условие2:
    блок кода
...
else:
    блок кода

 

При исполнении такого условного оператора сначала проверяется условие1. Если оно истинное, то исполняется блок кода под этим условием вплоть до выражения elif. Остальная часть конструкции игнорируется (Программа завершает работу с этой конструкцией). Если условие1 ложное, программа переходит к следующему выражению elif и проверяет условие2. Если оно истинное, то исполняется блок кода под этим условием вплоть до следующего выражения elif. И остальная часть условного оператора также игнорируется. Этот процесс продолжается до тех пор, пока не будет найдено условие, которое является истинным, либо пока больше не останется выражений elif, либо пока не дойдет до else. 

Ниже написан код, выполняющий все то же самое, что и сверху, только мы использовали в нем каскадный условный оператор:

grade = int(input('Введите вашу отметку: '))

if grade >= 90:
    print(5)
elif grade >= 80:
    print(4)
elif grade >= 70:
    print(3)
elif grade >= 60:
    print(2)
else:
    print(1)

Обратите внимание на выравнивание и выделение отступом, которые применены в инструкции if-elif-else

Структуры if-else и if-elif-else, взаимозаменяемы. Бывают случаи, где удобнее использовать первое и наоборот.

Заключительный блок else в операторе if-elif-else является необязательным:

grade = int(input('Введите вашу отметку: '))

if grade >= 90:
    print(5)
elif grade >= 80:
    print(4)
elif grade >= 70:
    print(3)
elif grade >= 60:
    print(2)
elif grade < 60:
    print(1)

Все три программы рассмотренные сегодня на уроке работают одинаково!
