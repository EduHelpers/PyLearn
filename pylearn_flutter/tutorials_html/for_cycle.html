Циклы

Компьютер может повторять одни и те же действия множество раз не уставая, за это отвечают циклы.

В Python существует две основных разновидности циклов:

  • циклы, повторяющиеся определенное количество раз for;
  • циклы, повторяющиеся до наступления определенного события while.

Цикл for 

Цикл for замечательно работает, если мы заранее знаем, сколько повторений (итераций) нам требуется сделать.

Рассмотрим код, который распечатает 10 раз "Hello, world!":

for i in range(10):
    print('Hello, world!')

Структура цикла for в Python выглядит так:

for название_переменной_цикла in range(количество_повторений):
    блок кода

Двоеточие (:) в конце строки с инструкцией for сообщает интерпретатору Python, что дальше находится блок команд. В блок команд входят все строки, расположенные с отступом в 4 пробела от строки с инструкцией for, вплоть до следующей строки без отступа. Блок команд, который выполняется в цикле for, называется телом цикла.

В прошлых урока мы вводили малое количество данных, обрабатывали их  и помещали в разные переменные. С помощью цикла for можно считывать и обрабатывать сколько угодно чисел (строк и т.д.). 

Рассмотрим следующий программный код:

for i in range(5):
    name = input()
    print("Имя участника:", name)

print("Цикл завершен")

Такая программа считывает 5 имен и выводит на их на экран. Поскольку вторая и третья строки выделены отступом, Python считает, что это тело цикла, которое выполняется 5 раз, а вот четвертая строка не является частью цикла (Так как нет отступа) и будет выполнена всего один раз после завершения цикла.

Теперь посмотрим, как работает переменная цикла.

for i in range(5):
    name = input()
    print("Имя участника №", i, ': ', name, sep='')

print("Цикл завершен")

Допустим входные данные были такие:

Саша
Маша
Даша
Ваня
Женя

Тогда программа выведет на экран следующее:

Имя участника №0: Саша
Имя участника №1: Маша
Имя участника №2: Даша
Имя участника №3: Ваня
Имя участника №4: Женя
Цикл завершен

Параметр sep склеивает все параметры переданные в print(), и мы пользуемся небольшой хитростью, передавая в print() параметр - строку ": ", которая обеспечивает нужный нам пробел между двоеточием и именем участника.
Обратите внимание на то, что счет в цикле for начинается с нуля, он все также выполняет 5 итераций, но переменная цикла принимает значения от 0 до 4!

Имя переменной цикла можно дать любое, главное, чтобы оно состояло из букв латинского алфавита (Есть исключение в виде имени - "_", нижнее подчеркивание используют в качестве имени тогда, когда хотят подчеркнуть, что переменная цикла не используется)

На самом деле, в функцию range() можно передавать 2(стартовая_граница, конечная_граница) или 3(стартовая_граница, конечная_граница и шаг) параметра, то есть структура цикла for выглядит так:

for название_переменной_цикла in range(стартовая_граница, конечная_граница, шаг):
    блок кода

Теперь мы можем переписать прошлые программы разными способами:

for i in range(1, 6):
    name = input()
    print("Имя участника №", i, ': ', name, sep='')

print("Цикл завершен")

Выведет:

Имя участника №1: Саша
Имя участника №2: Маша
Имя участника №3: Даша
Имя участника №4: Ваня
Имя участника №5: Женя
Цикл завершен

Или например такая вариация кода:

for i in range(1, 10, 2):
    name = input()
    print("Имя участника №", i, ': ', name, sep='')

print("Цикл завершен")

Выведет:

Имя участника №1: Саша
Имя участника №3: Маша
Имя участника №5: Даша
Имя участника №7: Ваня
Имя участника №9: Женя
Цикл завершен

Можно также использовать отрицательный шаг, но тогда поменяются местами первые два параметра:

for i in range(5, 1, -1):
    name = input()
    print("Имя участника №", i, ': ', name, sep='')

print("Цикл завершен")

Выведет:

Имя участника №5: Саша
Имя участника №4: Маша
Имя участника №3: Даша
Имя участника №2: Ваня
Имя участника №1: Женя
Цикл завершен
