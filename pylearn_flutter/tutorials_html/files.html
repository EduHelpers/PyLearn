Заметьте, что после очередного вызова readline() , следующий ее вызов будет печатать следующую строку.

Метод readlines() возвращает список строк из файла. (Строкой в файле считается объект заканчивающийся знаком '\n' )

Тогда следующий код:

file = open('example.txt', 'r')
text = file.readlines()
print(text)
file.close()

Выведет:

['Testing - FirstLine', 
    'Testing - SecondLine',
    'Testing - Third Line', 
    'Testing - Fourth Line', 
    'Testing - Fifth Line']

Таким образом мы можем итерироваться по строкам из текстового файла.

file = open('example.txt', 'r')
text = file.readlines()
for line in text:
    print(line)
file.close()

Выведет:

Testing - FirstLine
Testing - SecondLine
Testing - Third Line
Testing - Fourth Line
Testing - Fifth Line

Запись в файл

Скорее всего вы уже поняли, что для записи какой-то информации в файл, его нужно открыть не в формате 'r' (для чтения), а в формате 'w' (для записи).

Следующий программный код, открывает файл test.txt (Или создает, если такого не существует) и записывает в него строку "Hello, world!", а затем закрывает файл:

file = open('test.txt', 'w')
file.write("Hello, world!")
file.close()

Метод write() позволяет записывать любую строку в открытый файл. Запись в файл построчно выполняется посредством записи нужной вам строки с последующей записью '\n' — символа перевода строки. То есть, если вы хотите записать несколько строк кода, обязательно разделяйте их специальным символом '\n'.

file = open("test.txt", "w")
file.write("Люблю Python!\nЭто крутой язык!")
file.close()

То же самое можно было сделать, добавив желаемые для записи в файл строки в список, а затем с помощью цикла for каждую записать в файл:

lines = ['Я люблю Python!',
    'Это крутой язык!', 
    'Надеюсь я сдам экзамен']
f = open('test.txt', 'w')
for i in range(len(lines)):
    f.write(lines[i] + '\n')
f.close()

Не забываем разделять разные строки специальным символом '\n'.
