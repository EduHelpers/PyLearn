Числовые типы данных

Целочисленный тип данных

Целые числа в Python представлены типом данных int (integer - целое число).
 

n = 17 # целое число

m = -17 # целое число

k = 0 # целое число

Для преобразования строки в целое число, мы используем команду int():

num = int(input())  # преобразование введеной строки в целое число

То же самое можно сделать не вводя данные с клавиатуры:

n = int('12345')  # преобразование строки в целое число

Обратите внимание, что если входная строка не является целым числом (То есть выглядит как-то так "abcdefg"), то при преобразование произойдет ошибка.

n = int("abcdefg") # ERROR

Вещественные числа

Дробные (вещественные) числа в информатике называют числами с плавающей точкой.

Для представления чисел с плавающей точкой в Python используется тип данных float.

e = 2.71828  # вещественное число
pi = 3.1415  # вещественное число

В Python, когда вы используете операцию деления /, результат всегда будет числом с плавающей точкой, даже если оба числа являются целыми. Вот пример:

print(4 / 2)  # Результат: 2.0, а не 2

Заметьте, что разделение целой и дробной части осуществляется через точку, а не запятую!

Для преобразования строки к числу с плавающей точкой мы используем команду float():

num = float(input()) # преобразование считанной строки в число с плавающей точкой

То же самое можно сделать не вводя данные с клавиатуры:

n = float('1.2345')  # преобразование строки к числу с плавающей точкой

 Обратите внимание, что если входная строка не является вещественным числом (То есть выглядит как-то так "abcdefg"), то при преобразование произойдет ошибка.

n = float("abcdefg") # ERROR
