Целочисленная арифметика

Чтобы в Python создать переменную целого типа данных, нужно опустить кавычки при объявлении переменной. Рассмотрим следующий код:

num1 = 7            # num1 - это число
num2 = 10           # num2 - это число
num3 = num1 + num2  # num3 - это число
# num2 - это число, запись num2 += num1
# равносильна num2 = num2 + num1
num2 += num1       
print(num3)
print(num2)

В результате выполнения такой программы будет выведено:

17
17

Запомните, что числа обозначаются без кавычек, а строки — с кавычками.

Основные операции с числами

В языке Python, как и в математике, над числами можно совершать 4 основные операции (+, −, *, /) - сложение, вычитание, умножение и деление соответсвенно.

a = 3
b = 2

print(a + b)
print(a - b)
print(a * b)
print(a / b)

Результатом выполнения такой программы будут числа:

5
1
6
1.5

Порядок выполнения операций

Порядок выполнения операций в Python аналогичен порядку выполнения операций, которые вы изучали на уроках математики.

num1 = 2 + 3 * 4
num2 = (2 + 3) * 4

print(num1)
print(num2)

Результатом выполнения такой программы будут числа 14 и 20. 

Преобразование типов

Напишем программу, которая считывает два целых числа и выводит на экран их сумму. Следующий код тут не сработает:

num1 = input()
num2 = input()

print(num1 + num2)

Python считает, что в переменных num1 и num2 находится текст, поскольку команда input() по умолчанию считывает именно текст. Значит в переменные num1 и num2 требуется положить данные типа int, а значит мы должны применить функцию int():

num1 = int(input())
num2 = int(input())

print(num1 + num2)
