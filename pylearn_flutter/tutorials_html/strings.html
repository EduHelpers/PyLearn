Строковый тип данных

Теперь следует познакомиться со строковым типом данных str(string - строка).

Для создания строковой переменной в Python, мы должны заключить нужный текст в кавычки, а если данные вводятся с клавиатуры - просто считать их с помощью команды input() (Так как она, как мы помним, считывает данные и передает их в переменную в формате строк). В Python можно использовать как одинарные кавычки, так и двойные:

s1 = 'Python rocks!'
s2 = "Python rocks!"
s = input()  # переменная s имеет строковый тип str

Для задания пустой строки, мы используем две кавычки одинакового типа:

s1 = ''   # пустая строка
s2 = ""   # пустая строка
s3 = ' '  # строка состоящая из одного символа пробела
s4 = " "  # строка состоящая из одного символа пробела

В Python мы можем преобразовывать строки к числу, для этого мы используем функцию str():

Рассмотрим следующий программный код:

num1 = 1777    # целое число
num2 = 17.77   # число с плавающей точкой
s1 = str(num1) # преобразовали целое число в строку '1777'
s2 = str(num2) # преобразовали число с плавающей точкой в строку '17.77'

В Python для удобства работы со строками существуют множество функции. Разберем некоторые, наиболее важные, из них.

Функция len() предназначена для определения длины строки. Длина строки - количество символов в ней.

s1 = 'abcdef'
length1 = len(s1)               # считаем длину строки из переменной s1
length2 = len('Hello, world!')  # считаем длину введенной строки
print(length2)

выведет:

6
13
