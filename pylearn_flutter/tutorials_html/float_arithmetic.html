Вещественная арифметика

Язык Python предоставляет четыре основных арифметических оператора для работы с вещественными числами (+, −, *, /) и один дополнительный (** – для возведения в степень).

Следующая программа демонстрирует арифметические операторы:

a = 13.5
b = 2.0

sum = a + b
diff = a - b
prod = a * b
div = a / b
exp = a ** b

print(a, '+', b, '=', sum)
print(a, '-', b, '=', diff)
print(a, '*', b, '=', prod)
print(a, '/', b, '=', div)
print(a, '**', b, '=', exp)

В результате работы такой программы будет выведено:

13.5 + 2.0 = 15.5
13.5 - 2.0 = 11.5
13.5 * 2.0 = 27.0
13.5 / 2.0 = 6.75
13.5 ** 2.0 = 182.25
