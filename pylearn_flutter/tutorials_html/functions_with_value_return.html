Функции с возвратом значения

Функции могут не только принимать параметры и вызывать другие функции, они так же могут возвращать значения, причём любого типа. Значение возвращается туда, где функция была вызвана. Мы уже знаем много функций, возвращающих значения: len() возвращает длину строки, str() возвращает строку, в которую преобразовалось какое-то значение, range() возвращает последовательность.

Формат функции с возвращаемым значением такой:

    def функция():
        код
        return выражение

выражение после слова return будет отправлено туда, где функция была вызвана. В return можно задать любое выражение с любыми переменными. Рассмотрим пример:

    def dif(a,b):
        return a - b
    print(dif(5, 2))

Output:

    3

Как мы видим, значение, посчитанное внутри функции вернулось в аргумент функции print(), где мы и вызывали нашу функцию.

Также в return можно записывать несколько значений, в таком случае принимать их нужно через множественное присваивание:

    def func(a):
        return a + 5, a + 10, a + 20
    x,y,z = func(7)
    print(x, y, z)

Output:

    12 17 27
