Функции высшего порядка

Функции высшего порядка в Питоне построены на особом свойстве функций: их можно передавать в качестве аргумента. Рассмотрим пример:

def quad(a):
    return a**4
solve = quad
print(solve(2))

Output:

16

Здесь мы передали функцию в переменную и вызвали её через эту переменную. Тем самым через это свойство можно делать более обобщённые программные решения, и функции высшего порядка нам в этом помогут.

Функции высшего порядка - это функции, принимающие и/или возвращающие другие функции.

Рассмотрим пример функции высшего порядка:

def repeat(n, func):
    for i in range(n):
        func()
def f():
    print("Хеллоу мир манера крутит мир")
repeat(2, f)

Output:

Хеллоу мир манера крутит мир
Хеллоу мир манера крутит мир

Теперь посмотрим на функцию, возвращающую другую функцию:

def double(func):
    def inner(arg):
        return func(func(arg))
    return inner
def mult_five(x):
    return x * 5
double(mult_five)(3)

Output:

75

Функция double здесь принимает на вход другую функцию и дважды её вызывает, сначала от аргумента, а потом от значения функции от аргумента. double возвращает функцию, поэтому чтобы получить значение функции мы в круглых скобках указываем аргумент 3.
